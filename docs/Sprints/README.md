# Sprints do MVP

Organizacao das proximas etapas ate a entrega do MVP da cantina escolar.

## Regras

- Cada sprint deve entregar valor funcional.
- As dependencias devem seguir a ordem natural do produto.
- O foco do MVP e atender operacao simples, rapida e rastreavel.

## Sprint sugerida

1. Fundacao tecnica
2. Cadastro e estoque
3. Vendas rapidas
4. Relatorios e fiscal
5. Agendamento e rotina de compras
6. Ajustes finais do MVP
